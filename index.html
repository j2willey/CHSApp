📱 Install Camp Hi-Sierra

Install to your home screen to get the full app experience — works offline, launches full-screen, no browser bar.

On iPhone / iPad (Safari):
  1. Tap the Share button ⬆ at the bottom of Safari
  2. Scroll down and tap Add to Home Screen
  3. Tap Add in the top-right corner
Must be done from Safari — Chrome on iPhone cannot install apps.
On Android (Chrome):
  1. Tap the ⋮ menu in the top-right
  2. Tap Install app or Add to Home Screen
  3. Tap Install to confirm
On Desktop (Chrome / Edge):
  1. Click the install icon ⊕ in the address bar
  2. Or use the browser menu → Install Camp Hi-Sierra
